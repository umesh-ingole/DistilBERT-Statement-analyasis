## Key Implementation Details

### How Data Flows Through Training

```
1. Load Preprocessed Data
   data/train/ → Dataset (18,000 samples)
   data/validation/ → Dataset (2,000 samples)
   ↓
2. Create DataLoader
   batch_size=16
   shuffle=True (for training)
   ↓
3. For Each Epoch
   ├─ Training Phase (train_epoch)
   │  ├─ For each batch:
   │  │  ├─ input_ids: (16, 128) int64
   │  │  ├─ attention_mask: (16, 128) int64
   │  │  ├─ labels: (16,) int64
   │  │  ├─ Forward pass → logits: (16, 2)
   │  │  ├─ Compute loss
   │  │  ├─ Backward pass
   │  │  └─ Update weights
   │  └─ Track: loss, accuracy per batch
   │
   └─ Validation Phase (validate)
      ├─ For each batch (no gradients):
      │  ├─ Forward pass
      │  ├─ Compute metrics (accuracy, F1, AUC)
      │  └─ Track predictions
      └─ Save best model (if best validation accuracy)
   ↓
4. Save Results
   models/best_model/ → Best model weights
   outputs/metrics.json → Training history
```
